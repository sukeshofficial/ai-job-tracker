# AI Job Tracker

A smart AI-powered platform to:

- Track job applications
- Manage interview stages
- Analyze resumes
- Get ATS score
- Skill gap analysis
- Smart reminders

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind
- ShadCN

### Backend
- Spring Boot
- PostgreSQL
- JWT Auth

### AI Service
- FastAPI
- NLP
- Resume Analysis

### Databases
- PostgreSQL
- MongoDB
